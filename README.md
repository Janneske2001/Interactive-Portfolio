# Interactive Portfolio

A 3D interactive portfolio built with Three.js. I wanted to create something new.

## Features

- Interactive 3D project grid
- Hover animations
- Gyro effects on mobile
- 3D previews of models
- Project showcase system

## Tech Stack

- Three.js
- Vite
- JavaScript
- WebGL

## Development

Install dependencies:

npm install

Run locally:

npm run dev

I included a .bat script to automatically run this line in a terminal.
